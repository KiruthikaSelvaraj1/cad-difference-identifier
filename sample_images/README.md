# Sample Images

Place your sample CAD image pairs here for testing.

## Expected Format

- **File types**: JPG, JPEG, or PNG
- **Content**: CAD drawings (white background, black/colored lines)
- **Naming convention** (suggested):
  - `drawing_v1.png` — Reference/original version
  - `drawing_v2.png` — Modified/comparison version

## Tips for Best Results

1. Use images from the same CAD project (same drawing, different revisions).
2. Images don't need to be exactly the same resolution — the system will resize automatically.
3. Slight rotational or positional shifts are handled by the ORB-based alignment step.
4. For best results, export CAD drawings at consistent DPI settings.
